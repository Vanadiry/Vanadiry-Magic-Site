收集到的建议

这里记录从各个途径收集到的建议，并实时更新执行状态，已经完成或已被拒绝的建议在这里归档。非常感谢各位的宝贵意见。

更多指令

这是一组特别的建议，由Geticer提交，关于这些建议将会分别报告状态

未知：/gamemode 0 == 关闭自动滚动（生存模式）
未知：/gamemode 3 == 打开自动滚动（旁观模式）（如果能让游戏忽略用户输入就更好了，变成真正的旁观模式）
拒绝：/kill == 直接bad ending（算成就吗）
正在进行：/stop 或 /quit == 直接退出
完成：/help == 傲娇的拒绝显示帮助，或者随机显示小贴士
未知：/summon == 充满“有限可能性”的召唤功能（比如/summon creeper == aww man）
拒绝：//set 0 == 进入特殊剧情：一百条空白对话（因为被清空了）
未知：另外还可以考虑增加一些经典的作弊码（如panzer、painkiller等）
列入计划：/tp （地点） == 触发特殊剧情：传送到对应位置的剧情
完成：在一些“没有提示真的能卡关”的地方输入/help可获得提示，比如 1:50 处输入/help，获得提示：“如果环境音效太吵的话，扔掉不就好了？”

复杂类型

这是一组特别的建议，由懒惰的105提交，关于这些建议将会分别报告状态

未知：在游戏刚开始的时候我们扮演碳碳当碳碳被店员推进游戏后，才正式开始可以正常攻略角色，路线分为6条路线碳碳、花花、沃玛普通路线、沃玛秘密线、怒九笑小弟路线（被击败）、怒九笑老大路线（击败怒九笑）
列入计划：没有击败怒九笑每周需要交保护费，如果击败了，就不用交了。
列入计划：对了，我还有个想法一周末碳碳被推荐游戏后，我们不会办演碳碳，这时候才可以攻略碳碳，到了二周目后碳碳，再次被推进游戏后，我们仍然会扮演碳碳，这是我们的操作的人物就变成了碳碳，碳碳可以攻略的路线有两条一条是花花，第二条就是我们一周末扮演的主角，由于碳碳的人设是女的所以攻略花花就会达，百合结局
未知进了怒九线，可以成为怒九的小弟，不用交保护费了成为小弟后可以开启RPG玩法，去找其他学生收保护费，不过由于还RPG玩法，所以要加一些饭店可以恢复体力，药店恢复血量，运动店购买装备。
正在进行：在游戏文件里藏个花花的这最初设定文件，怎么样？
列入计划：在游戏还没有开始的时候，直接把花花的人物文件给删掉，这样碳碳就不会被推进游戏里，怎么样？
