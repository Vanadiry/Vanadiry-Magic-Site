iOS/iPadOS 本机存档/应用数据迁移

须知

  1. 仅适用于iOS/iPadOS
  2. 迁入存档的机器需要该设备的Apple ID及密码
  3. 可能有数据丢失风险，请提前备份
  4. 需要迁移数据的软件，需要安装相同版本，或者迁出数据的软件版本低于迁入数据的软件版本。否则可能数据异常。
  5. 需要一台安装有iMazing的电脑，Windows与MacOS均可。（iMazing官网）
  6. 如果你的设备不是从官方渠道购买，则可能是“越狱过锁”或者“黑解”设备，由于执行恢复步骤时，设备会进入激活界面，所以这些设备将会无法激活。

第一步 备份应用并提取数据

  1. 连接需要迁出数据的机器到电脑。
  2. 点击“管理应用程序”。（这一步可能需要输入Apple ID密码）
  3. 找到需要备份的软件，右键选择“备份应用程序数据”。
  4. 选择提取目标文件夹，然后在提取选项中选择“备份并提取应用数据”，最后点击下一步。
  5. 等待备份和提取数据完毕。
  6. 此时在刚才选择的目标文件夹中会生成一个扩展名为imazingapp的文件，请妥善保存。

第二步 恢复数据到新设备

  1. 连接需要迁入数据的机器到电脑。
  2. 点击“管理应用程序”。（这一步可能需要输入Apple ID密码）
  3. 找到需要恢复数据的软件，右键选择“恢复应用程序数据”。
  4. 选择刚才的imazingapp文件，然后点击完成。
  5. 如果提示设备可能重启，请点击继续。
  6. 按照屏幕提示在手机上操作，关闭设备的“查找”功能。（这一步需要输入Apple ID密码）
  7. 关闭“查找”后，恢复会继续进行，直到显示“已安装”。

第三步 激活设备

  1. 转到你的设备继续操作
  2. 设备会显示“恢复完成”，请点击继续
  3. 在应用和数据步骤中选择“不要转移应用和数据”
  4. 继续，直到进入桌面

至此，数据迁移就完成了。
